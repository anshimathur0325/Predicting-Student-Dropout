Student Drop Out Prediction- Exploratory Data Analysis

This website predits whether someone will drop out or graduate based on the input features.

In order to choose these features, I conducted exploratory data analysis and feature selection.

The heatmap below shows the relationships between the features of a dataset of students.

The features that were chosen are the ones that had the highest correlation with the target variable. The number of credits someone was taking seemed to have the most impact, which was surprising at first.

The violin plot below shows the relationship between the features and the target variable. Notice the difference between the distributions of students that dropped out vs. graduated.

It was important to me that the input features were easy to understand and interpret. For this reason, I dropped features that a common user of this website would be unable to understand. While this may affect the accuracy of the model, it resulted in a more usable interface.

This model was trained using a Random Forest Classifier and achieved an accuracy of .90.
